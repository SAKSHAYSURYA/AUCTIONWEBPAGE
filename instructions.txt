-----------------------------> STEPS TO RUN THE PROJECT <------------------------------------

. After you finish downloading the project, unzip the project file and head over to your XAMPP directory.
There you’ll find a folder naming “htdocs”.

. Inside the “htdocs” folder, paste the project folder (not the .zip one, but the extracted one).

. Open your  browser; we recommend using Google Chrome.

. Then, go to URL “http://localhost/phpmyadmin“.

. Create a Database with a name that is provided inside the “dbmsmpdb”.


. Finally, click on the “Import” tab and choose the database file (dbmsmpdb.sql) which is provided under the folder named “SQL”.

. After setting up all these, go to URL “http://localhost/auction-webpage/"

. Abd it is ready to go.
